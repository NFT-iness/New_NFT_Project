UNDERSTANDING NFT-INESS

Aim behind NFT-INESS

We present the NFT-iness project, an exploration of the activity on the Ethereum non-fungible token (NFT) market from 2017 and early 2023 constructed from the API of Etherscan.

We then consider the following questions:
- Who owns NFTs on the Ethereum blockchain? Are NFTs being traded in a bubble? We explore this idea in the 'commerce of NFT' section of the website.
- What are the trends in NFT market? What is the growth pattern of this market? We explore this idea in the 'trends in NFT' section of the website.

DATA DESCRIPTION

The data used in the project is acquired using an API of Etherscan (a Block Explorer and Analytics Platform for Ethereum) we were able to obtain Transaction Data.

Restrictions/Limits:
- The provided API has a hyperparameter called "offset" which is limited to the value: 10000. Using the maximum value we were able to obtain over 25000 transactions reaching back till 2017. To make a visual analysis more easier to perform we decided to create three Datasets that vary at the "offset" parameter.

--- NFT_RawData_Offset100
--- NFT_RawData_Ofsset1000
--- NFT_RawData_Ofsset10000

For furture details about the data, download the documentation Jupyter notebook and to access our scripts and data, head over to the github repo of the project
